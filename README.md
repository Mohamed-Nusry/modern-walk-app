# modern-walk-app

This is a sample design for a fashion retail web app called "Modern Walk".

## Pre Requires
### Node v18 or up

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Test

```sh
npm test
```

### Lint

```sh
npm run lint
```
